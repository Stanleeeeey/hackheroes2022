# hackheroes2022

jak odpalić?

otwórz terminal w folderze hackheroes2022

użyj komendy

`$ pip install requirements.txt`

by zainstalować biblioteki

oraz komendy

`$ python run.py`

by urochomić serwer

serwer powinien zacząć działać na localhoście i jako development server 

w razie problemów prosimy napisać
